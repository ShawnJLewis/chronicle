# Chronicle · 编年史

一个基于 Python 的文字冒险游戏引擎，带有完整的剧情分支系统。

## 游戏介绍

**编年史 · 雾中来信** — 你收到一封没有署名的信，信封里是一张灯塔的照片。十年前那座灯塔在一场风暴中坍塌，三人遇难，其中包括你的父亲。现在，有人告诉你：那座灯塔下面有一个房间，有人在等你。

玩家通过选择推进剧情，体验一个关于记忆、归还与等待的故事。

### 剧情概述

- **19 个章节**，2 个不同结局
- 选择影响道具获取和剧情走向
- 自动存档系统，随时可继续

## 环境要求

- Python 3.8+
- 无其他依赖（仅使用 Python 标准库）

## 运行方式

```bash
cd chronicle
python3 game.py
```

## 操作说明

- 输入数字选择选项
- 输入 `q` 或 `quit` 保存并退出
- 游戏自动保存进度

## 项目结构

```
chronicle/
├── game.py      # 游戏引擎
├── story.json   # 剧情数据
└── README.md
```
